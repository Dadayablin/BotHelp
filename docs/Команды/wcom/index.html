Для всех

Все команды расположены в алфавитном порядке (английском).​

ping​

(откл/вкл. %ping)

Проверка бота на активность.

%ping | !ping

Статусы для afk​

afk​

Задаёт вам статус "afk".

text - любая фраза (не обязательно).

%afk text

brb​

Задаёт вам статус "brb".

text - любая фраза (не обязательно).

%brb text

Задаёт вам статус "chill".

text - любая фраза (не обязательно).

%fishing text

Удаляет ваш статус при написании в чат​

rafk​

Возвращает вам последный статус.

У вас есть 5 минут после выхода из "afk".

%rafk

remind​

Оповестит человека, когда он напишет в чат.

user - ник человека.

text - текст, который нужно передать.

%remind user text

remindme​

Оповестит вас через определённое количество времени.

time - время, через которое вас оповестить (указывать в секундах).

text - текст, с которым вас оповестят.

%remindme time text

tuck​

(откл/вкл. %tuck)

Укладывает в кровать человека.

user - ник человека (если не указать, то уложишь сам себя).

%tuck user

followage | fa​

(откл/вкл. %fa)

Показывает длительность подписки на канал.

user - ник человека (если не указать, то выведет вашу длительность).

channel - название канала (если не указать, то выведет на данном канале).

%followage | %fa user channel

followers​

(откл/вкл. %followers)

%followers user

(откл/вкл. %flu)

isbanned​

(откл/вкл. %isbanned)

%isbanned user

isbot​

(откл/вкл. %isbanned)

%isbot user

subage | sa​

(откл/вкл. %sa)

Показывает длительность подписки на канал.

user - ник человека (если не указать, то выведет вашу длительность).

chan - название канала (если не указать, то выведет на данном канале).

Если не указывать, то покажет длительность вашей подписки.

%subage | %sa user chan

user | u​

(откл/вкл. %u)

Показывает информацию о человеке.

user - ник человека (если не указать, выведет информацию о вас).

-id - если хотите узнать информацию по id (тогда вместо ника нужно указывать id).

%user | %u user -id

%vanishme | %vanish

setloc​

(откл/вкл. %погода)

Устанавливает вашу локацию.

loc - Ваша локация.

%setloc loc

Также можно включить режим инкогнито (никто не будет видеть ваше местоположение)

show - стандартный режим.

inc - инкогнито.

%setloc inc | show

throw | кинуть​

translate | t​

(откл/вкл. %t)

Переводит фразу.

from - язык входных данных.

to - язык данных при выходе.

text - сам текст.

%translate | %t from to text

(откл/вкл. %погода)

place - Ваш город. (можно не указывать)

%weather | %погода | %w place
