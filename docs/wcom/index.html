Команды

Основные функции​

afk​

Задаёт вам статус "afk".

text - любая фраза (не обязательно).

%afk text

chill​

Задаёт вам статус "chill".

text - любая фраза (не обязательно).

%chill text

brb​

Задаёт вам статус "brb".

text - любая фраза (не обязательно).

%brb text

fishing​

Задаёт вам статус "fishing".

text - любая фраза (не обязательно).

%fishing text

food​

Задаёт вам статус "food".

text - любая фраза (не обязательно).

%food text

gn​

Задаёт вам статус "gn".

text - любая фраза (не обязательно).

%gn text

study​

Задаёт вам статус "study".

text - любая фраза (не обязательно).

%study text

tilt​

Задаёт вам статус "tilt".

text - любая фраза (не обязательно).

%tilt text

work​

Задаёт вам статус "work".

text - любая фраза (не обязательно).

%work text

Удаляет ваш статус при написании в чат​

rafk​

Возвращает вам последный статус.

У вас есть 5 минут после выхода из "afk".

%rafk

remind​

Оповестит человека, когда он напишет в чат.

user - ник человека.

text - текст, который нужно передать.

%remind user text

remindme​

Оповестит вас через определённое количество времени.

time - время, через которое вас оповестить.

text - текст, с которым вас оповестят.

%remindme time text

tuck​

(откл. %tuck)

Укладывает в кровать человека.

user - ник человека (если не указать, то уложишь сам себя).

%tuck user

Другие команды​

анекдот​

(откл. %анекдот)

Выведет вам случайный анекдот.

%анекдот

ping​

(откл. %ping)

Проверка бота на активность.

%ping | !ping

7tvpfp​

(откл. %7tvpfp)

Выведет вам аватарку человека с сайта 7tv.

%7tvpfp

accage​

(откл. %accage)

Покажет возраст вашего аккаунта.

%accage

cat​

(откл. %cat)

Отправит в чат случайное фото кота.

%cat

catfact​

(откл. %catfact)

Выведет случайный факт о котах.

%catfact

chattercount | cc​

(откл. %cc)

Показывает количество участников чата на данный момент.

user - ник человека (если не будет, то выведет у вас).

%chattercount user | %cc user

сolor​

(откл. %color)

Скажет какой цвет у человека.

user - ник человека.

%color user

currency | cur​

(откл. %cur)

Переводит валюту.

value - количество валюты.

from - первоначальная валюта.

end - получаемая валюта.

(to обязательно)

%currency | %cur value from to end

followage | fa​

(откл. %fa)

Показывает как давно вы зафолловились на канал.

%followage | %fa

followers​

(откл. %followers)

Показывает количество фолловеров на вашем канале.

%followers

google​

(откл. %google)

Выводит ссылку по запросу.

text - любая фраза.

%google text

lastmessage | lm​

(откл. %lm)

Выводит последнее сообщение человека.

user - ник человека (если не указать, то выведет ваше последнее сообщение).

%lastmessage | %lm user

pick​

(откл. %pick)

Выберет один из вариантов.

v1 v2 v3 ... - варианты для выбора

%pick v1 v2 v3 ...

pyramid | пирамида | pmd​

(откл. %pyramid)

Строит пирамиду из символов (их должно быть больше 3);

text - любые символы.

%pyramid | %пирамида | %pmd text

randomcolor | rcolor​

(откл. %rcolor)

Выведет случайный цвет случайного человека

%randomcolor | %rcolor

steam​

(откл. %steam)

Информация о steam.

avatar​

Выведет ссылкой аватарку человека в steam.

id - айди человека.

%steam avatar id

id​

Выведет ваш id по ссылке на аккаунт.

text - ваша ссылка на аккаунт steam.

%steam id text

players​

Выводит количество игроков сейчас в игре.

id - id игры.

%steam players id

info​

Выводит информацию об игре по id.

id - id игры.

%steam info id

subage | sa​

(откл. %sa)

Показывает длительность подписки на канал.

user - ник человека.

chan - канал.

Если не указывать, то покажет длительность вашей подписки.

%subage | %sa user chan

translate | t​

(откл. %t)

Переводит фразу.

from - язык входных данных.

to - язык данных при выходе.

text - сам текст.

%translate | %t from to text

user | u​

(откл. %u)

Показывает информацию о человеке.

user - ник человека(если не указать, выведет информацию о вас).

%user | %u user

weather | погода | w​

(откл. %погода)

Выводит информацию о погоде.

place - Ваш город.

%weather | %погода | %w place

user - ник человека (если не указать, будет выбран случайно).

%fuck | %послать user

%hug | %обнять user

kiss | поцеловать​

mama | мама​

user - ник человека (если не указать, будет выбран случайно).

%meet user
